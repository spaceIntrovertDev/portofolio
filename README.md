# Philipp Wiedemann – Personal Portfolio

A modern, responsive personal portfolio built with **Next.js 14 (App Router)**, **React**, and **TypeScript**. Features a persistent sidebar (mobile: fixed top navbar), dark mode, interactive project modals, and full CV integration.

## Features

- **Responsive Layout**: Sidebar on desktop → fixed top navbar on mobile (visible on scroll)
- **Dark Mode**: Native Tailwind support
- **Interactive Projects**: Click images for full-screen modal viewer
- **Performance Optimized**: Next.js Image, static generation, Tailwind JIT
- **SEO Friendly**: Metadata, Open Graph tags
- **Print-Ready**: All pages formatted for PDF export

## Sections

- **Home** – Profile card with photo and summary
- **Experience** – Professional history (RESERViSiON GmbH, Workwise GmbH)
- **Education** – Bachelor’s in Applied Computer Science + academic projects
- **Skills** – Programming, frameworks, tools, and soft skills
- **Projects** – Academic + self-learning (Twitter Analysis, Chatbot, Multi-Tenant Sales Dashboard)
- **Contact** – Address, email, GitHub, CV download
- **About This Site** – Tech stack, design decisions, future plans

## Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/portfolio.git
cd portfolio

# Install dependencies
npm install

# Run locally
npm run dev
# Open http://localhost:3000
```
